Bem-vindo à Calculadora de Formas Geométricas!

Esta calculadora foi projetada para ajudá-lo a calcular a área de várias formas geométricas simples. Selecione uma forma geométrica no menu acima para começar.

Instruções para calcular áreas:

Quadrado:

Para calcular a área de um quadrado, multiplique o comprimento de um dos lados pelo próprio lado.

Área = lado * lado

Triângulo:

Para calcular a área de um triângulo, multiplique a base do triângulo pela altura e divida o resultado por 2.

Área = (base * altura) / 2

Trapézio:

Para calcular a área de um trapézio, some as medidas das bases, multiplique pela altura e divida o resultado por 2.

Área = ((baseMaior + baseMenor) * altura) / 2

Círculo:

Para calcular a área de um círculo, multiplique o quadrado do raio pela constante π (pi).

Área = π * raio * raio

Valor aproximado de π (pi): 3.14159

Exemplos:

Exemplo 1: Quadrado

Suponha que temos um quadrado com um lado de comprimento 5 unidades.

Área = 5 * 5

Área = 25 unidades quadradas

Exemplo 2: Triângulo

Suponha que temos um triângulo com base de comprimento 8 unidades e altura de 6 unidades.

Área = (8 * 6) / 2

Área = 24 unidades quadradas

Exemplo 3: Trapézio

Suponha que temos um trapézio com bases de comprimento 6 e 10 unidades e altura de 4 unidades.

Área = ((6 + 10) * 4) / 2

Área = 32 unidades quadradas

Exemplo 4: Círculo

Suponha que temos um círculo com raio de 3 unidades.

Área = 3.14159 * 3 * 3

Área ≈ 28.27431 unidades quadradas
